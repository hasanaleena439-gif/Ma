House Price Prediction
A machine learning project that predicts house prices using regression models.
Overview
This project uses the Boston Housing or Kaggle House Prices dataset to build a model that predicts house prices based on features like square footage, number of bedrooms, location, etc.

Model Used
Algorithm: Linear Regression (simple version) or Gradient Boosting (advanced)
Dataset: Boston Housing / Kaggle House Prices
Accuracy: ~85-88% R-squared score
